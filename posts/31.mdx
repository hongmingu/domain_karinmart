MacBook에서 라우터의 IP 주소를 확인하고 MAC 주소를 변경하려면 터미널을 이용하면 된다.

## 라우터의 IP 주소 확인
대부분의 경우 라우터의 IP 주소는 기본 게이트웨이로 설정되어 있다. 

이를 확인하려면 아래의 명령어를 터미널에 입력하면 된다.
```
bash 코드
netstat -nr | grep default
이 명령어를 실행하면 기본 게이트웨이의 IP 주소를 확인할 수 있다.
```
## MAC 주소 변경
먼저 현재의 MAC 주소를 확인하려면 다음 명령어를 터미널에 입력하자.
```
ifconfig en0 | grep ether
```
여기서 en0는 대게 무선 네트워크 인터페이스를 가리킨다. 이더넷을 사용하는 경우 en1 또는 다른 값을 사용할 수 있다.

MAC 주소를 변경하려면 sudo를 사용하여 ifconfig 명령어를 다음과 같이 실행하자.
## 맥주소 변경
```
sudo ifconfig en0 ether xx:xx:xx:xx:xx:xx
```
여기서 xx:xx:xx:xx:xx:xx는 원하는 MAC 주소를 입력하면 된다.

MAC 주소 변경은 네트워크 안정성 및 보안 문제를 일으킬 수 있으므로 주의가 필요하다. 항상 원래 MAC 주소를 기록해 두고 필요한 경우 복구할 수 있도록 해 놔야 한다. 이 명령어는 재부팅 후에 원래대로 돌아올 수 있기 때문에 MAC 주소를 영구적으로 변경하려면 추가적인 설정이 필요하다.

또한 라우터의 설정이나 네트워크 상황에 따라 위의 명령어가 제대로 작동하지 않을 수 있으니 참고할 것. 기능이 제대로 동작하는지 확인하려면 항상 테스트를 해보는 것이 좋음.
