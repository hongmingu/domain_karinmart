이 게시물에선 Zsh의 명령어와 Oh My Zsh에 대해 알아볼 것이다.

## 명령어 입력과 히스토리
Zsh에서 명령어는 기본적으로 터미널에 입력하여 실행한다. Zsh는 입력한 명령어의 히스토리를 저장하여, 화살표 키(↑, ↓)를 이용해 이전에 사용한 명령어를 쉽게 볼 수 있다. 또한 Ctrl + r을 통해 히스토리 검색 기능을 제공, 사용자가 이전에 입력했던 명령어를 빠르게 찾을 수 있다.

Zsh의 히스토리는 기본적으로 ~/.zsh_history에 저장된다.

### 히스토리 검색 사용법
```
# 히스토리에서 "git"이라는 문자열을 포함하는 명령어 검색
Ctrl + r (그 후 "git" 입력)
```
## 파일과 디렉터리 탐색
Zsh는 확장된 글로브 기능을 제공한다. 예를 들어 **/*.txt 패턴을 사용하면 모든 하위 디렉토리에서 .txt 확장자를 가진 파일들을 찾을 수 있다. 또한, cd 명령어의 경우 부분적으로 디렉토리 이름을 입력하면 자동 완성 기능을 통해 전체 경로를 빠르게 찾을 수 있다.

### 확장된 글로브 기능 사용법
```
# .txt 파일 전체 검색
ls **/*.txt
```
## 명령어 치환 및 자동 완성 기능
$(command) 또는 `command` 형식으로 명령어의 출력 결과를 다른 명령어의 입력으로 사용할 수 있다. 자동 완성 기능은 특히 Zsh의 강력한 기능 중 하나로, 명령어나 파일, 디렉토리 이름의 일부를 입력하고 Tab 키를 누르면 관련된 후보들을 보여줍니다.
### 명령어 치환 사용법
```
# 현재 디렉토리의 파일 개수 출력
echo "There are $(ls | wc -l) files in the current directory."
```
### 자동 완성 사용법
```
git st[TAB] 
# 위를 입력하면 "git status"로 자동 완성된다.
```
## 테마와 플러그인
Zsh는 사용자가 터미널 환경을 개인화할 수 있도록 다양한 테마와 플러그인을 지원한다. 이를 통해 프롬프트의 디자인을 변경하거나 추가 기능을 확장할 수 있다.

## Oh My Zsh 소개
Oh My Zsh는 Zsh의 환경 설정을 관리해주는 프레임워크이다. Oh My Zsh를 사용하면 Zsh의 설정, 테마, 플러그인을 쉽게 관리할 수 있다.

### Oh My Zsh 설치
```
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

## 주요 테마 및 플러그인 소개
### 테마
 agnoster, robbyrussell, af-magic 등.
### 테마 변경 사용법
```
# ~/.zshrc 파일을 열고 
ZSH_THEME="agnoster"
```

### 플러그인
git, docker, z, extract 등.

## 테마 및 플러그인 설치 및 설정 방법
Oh My Zsh가 설치된 상태에서는 ~/.zshrc 파일에서 테마나 플러그인 설정을 변경할 수 있다. 테마를 변경하려면 해당 파일에서 ZSH_THEME 값을 원하는 테마 이름으로 변경하면 된다.
플러그인 설치 사용법 (z 플러그인):
```
# ~/.zshrc 파일을 열고 
plugins=(... z)
```
이후 쉘을 재시작하면 변경사항이 적용된다.
## 고급 기능
Zsh는 스크립팅, 명령어 치환, 확장된 글로브 기능, 배열 및 연관 배열 지원 등 많은 고급 기능을 제공한다. 이를 활용하면 복잡한 작업도 쉽게 자동화하거나 커스터마이징할 수 있다.
### 배열 사용법
```
fruits=("apple" "banana" "cherry")
echo $fruits[1]  # apple 출력
```

### 연관 배열 사용법
```
declare -A person
person[name]="John"
person[age]=30

echo $person[name]  # John 출력
```
