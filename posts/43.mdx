## Roda 소개
Roda는 라우팅 트리를 사용하는 루비 웹 프레임워크이다.

#### 간결
Roda의 코드는 매우 간결하며, 라우팅을 트리 구조로 정의한다.
#### 속도
작은 오버헤드로 높은 성능을 제공한다.
#### 확장성 
다양한 플러그인을 통해 기능을 확장할 수 있다.

#### Roda와 다른 웹 프레임워크와의 비교
##### Sinatra
Roda는 Sinatra와 유사한 간결함을 제공하지만 라우팅 트리를 사용하여 더 구조적인 방식으로 라우팅을 정의한다.
##### Rails
Rails보다 덜 복잡하고, 더 적은 특징을 가지고 있어 작고 빠른 프로젝트에 적합하다.
## 2. 설치 및 프로젝트 생성
### Roda 설치하기
Roda를 설치하려면 먼저 Ruby가 시스템에 설치되어 있어야 한다. 그리고 gem을 사용해 Roda를 설치한다
```
gem install roda
```
### 새 프로젝트 생성하기
새로운 디렉토리를 만들고 Gemfile을 생성한다. Gemfile에 다음 내용을 추가한다.
```
source 'https://rubygems.org'
gem 'roda'
```
그 다음 Bundler를 사용하여 gem을 설치한다
```
bundle install
```
### 기본 앱 생성
app.rb라는 파일을 생성하고 다음 내용을 추가한다
```
require 'roda'

class App < Roda
  route do |r|
    r.root do
      'Hello, Roda!'
    end
  end
end
```
이 코드는 기본 라우트를 정의하고 루트 URL로 요청을 받으면 "Hello, Roda!"라는 문자열을 반환한다.

### 서버 실행
config.ru 파일을 생성하고 다음 내용을 추가한다
```
require './app'
run App
```
그 다음에 명령 프롬프트에서 다음 명령을 실행하여 서버를 시작한다
```
rackup
```
이제 웹 브라우저에서 로컬호스트:9292 에 접속하면 "Hello, Roda!"라는 메시지를 볼 수 있다.

이렇게 간단한 과정을 통해 Roda를 설치하고 기본 웹 애플리케이션을 실행할 수 있다. 
Roda는 이러한 기본 구조를 기반으로 복잡한 웹 애플리케이션을 구축할 수 있도록 확장할 수 있게 해준다.
